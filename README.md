# 20-智慧社区管理平台

[文档地址](http://wechat.zjrcsy.cn/)

#### 介绍

技术：
基于 B/S 架构 SpringBoot+MySQL+Vue

环境：
Idea mysql maven jdk1.8

软件功能

主要功能描述
（1）统计分析图表:统计居住率
（2）物业管理 :小区管理 居民管理  地图展示 
（3）门禁管理 :人脸识别照片比对出入居民出入记录  访客记录
（4）系统管理 : 用户的管理   角色 菜单  日志 等
